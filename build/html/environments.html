class environments.Two_Sensors(mu=0.1, m=2.5, g=9.8, v=1, max_reward=1)¶

Bases: object

This class contains the environment for the experiment II.

Parameters

  • mu (float, optional) – Friction coefficient between the box and the floor. , defaults to 0.1

  • m (float, optional) – Mass of the bullet in Kg, defaults to 2.5

  • g (float, optional) – Acceleration of gravity in m/s^2, defaults to 9.8

  • v (int, optional) – Velocity of the bullet in m/s, defaults to 1

  • max_reward (int, optional) – Maximum allowed reward for the agent, defaults to 1

give_reward(prediction, observed, sigma=0.05)¶

Generates the reward based on a Gaussian function of the relative difference between the predicted and the observed value.

Parameters

  • prediction (float) – predicted value

  • observed (float) – real observed value

  • sigma (float, optional) – Width of the Gaussian. The smaller sigma, the higher the needed precision to get a reward, defaults to 0.05

Returns

Obtained reward

Return type

float / numpy array

test_shooting()¶

Samples uniformly at random a velocity for the bullet from the range [v-0.5*v,v+0.5*v], where v is the velocity of the bullets in the initial shooting. Then it calculates the position of the box after being shot with the random velocity.

Returns

v_test (sampled velocity), d_test (corresponding distance)

Return type

float, float
