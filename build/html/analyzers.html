analyzers module¶

class analyzers.Two_Layers_single_output(input_size, learning_rate=0.001)¶

Bases: object

This class is just a two-layer Keras neural network to process the data taken by the experimenter to produce a single output.

Parameters

  • input_size (int) – Number of inputs feeded to the analyzer

  • learning_rate (float, optional) – Gradient descent (Adam) learning rate, defaults to 0.001

build_model()¶

Uses keras to build de model. You should modify here the code to change the neural network architecture. Default to a input_sizex16x16x1 fully connected sequential network.
